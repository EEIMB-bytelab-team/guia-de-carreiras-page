# Guia de Carreiras — Orientação para Estudantes do Ensino Médio
Este repositório contém um site desenvolvido para ajudar estudantes do ensino médio a descobrirem possíveis caminhos profissionais e a fazerem escolhas mais conscientes sobre seu futuro.


# Objetivo do Projeto

O site foi criado para:

• Apresentar diversas opções de carreiras de forma clara e acessível.

• Ajudar estudantes a entender habilidades e áreas de interesse relacionadas a cada profissão.

• Oferecer informações básicas sobre mercado de trabalho, formações necessárias e áreas de atuação.

• Apoiar o processo de escolha profissional, incentivando autoconhecimento e planejamento.


# Tecnologias Utilizadas

O projeto foi desenvolvido com:

• HTML5 — estrutura e conteúdo das páginas
 
• CSS3 — design, estilo e identidade visual

• JavaScript (JS) — interatividade e funcionalidades dinâmicas


 #  Estrutura Geral do Projeto

│── index.html -  Página principal

│── style.css - Estilos do site

│── script.js - Funções e interações

└── README.md -  Documentação do projeto


 # Funcionalidades do Site

• Lista organizada de carreiras e áreas profissionais.

• Descrições simples sobre cada profissão.

• Informações sobre cursos, trilhas de formação e áreas de atuação.

• Interface pensada para estudantes do ensino médio.


 # Público-Alvo

- Estudantes do ensino médio

- Professores e responsáveis interessados no processo de escolha profissional
  

 # Impacto Educacional
 
 Este site contribui para:

- Incentivar o autoconhecimento dos estudantes.

- Facilitar o acesso a informações sobre carreiras.

- Apoiar decisões mais conscientes sobre o futuro.

- Criar um espaço simples e confiável de orientação profissional.
